## 项目简介
为了方便实验室药品的记录和存取，我们搭建了一个网站用于实验人员使用，主要包括支持药品信息的数字化录入、分类存储、实时库存查询及使用记录追踪。  

**平台链接**：[https://123s-oss.github.io/2/](https://123s-oss.github.io/2/)
